https://roadmap.sh/projects/task-tracker
Task Tracker is a lightweight command-line application built with Python that helps you manage   
  your daily tasks efficiently. It allows you to add, update, and delete tasks, as well as mark
  them as in-progress or done. All tasks are stored locally in a `tasks.json` file that is created 
  automatically if it does not exist. You can list all tasks or filter them by status: todo,       
  in-progress, or done. The app uses only Python's built-in libraries with no external dependencies
   required. Simply run `python taskTracker.py <command>` in your terminal to get started.
